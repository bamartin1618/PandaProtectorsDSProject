The Effect of COVID Safety Measures on the Impacts of COVID

Our project studied a dataset consisting of various safety measures taken to mitigate the COVID-19 pandemic and also the impacts of the pandemic and wanted to determine to what extent did the effectiveness and implementation of the safety measures have on the impacts of COVID.

Hypothesis: If COVID safety measures are more effective, then the impacts of COVID will be less severe.

We first installed Plotly and imported the panda module. Using the panda module, we read the csv file of our raw data, and defined this as our dataset (variable name). Next, we cleaned this dataset and redefined our dataset variable after keeping the uncleaned set as our original dataset. Finally, we selected from the now cleaned dataset the categorical columns we wanted to analyze. We created a new set with just these columns, naming it as the new condensed dataset.

"""
We want a function such that we can pass in a column name (percentage of vaccinated)

The function we used for total number of deaths took the counties in each state, added up the deaths, and converted it from total number of deaths
for each county to total number of deaths for each.

We want to do the same thing, but we want our function to take in three inputs. The first input is our dataset, the second input is our column name, the third input describes how we're grouping columns.

For total number of deaths, we were grouping columns by using sum() which just added the values in the num_deaths column. But we could use functions like mean or mode.

Input: dataset, column_name, operation we perform on the column (sum, mean, mode, etc.)
Output: Bar Graph
"""

def visualize_state_data(dataset, column_name, column_operation):
    
    bar_graph_data = dataset[['state', column_name]]
    
    bar_graph_data = bar_graph_data.dropna(0, how='any')
    
    bar_graph_data = eval('bar_graph_data.groupby(\'state\').' + column_operation + '()')
    
    bar_graph_data['state'] = bar_graph_data.index
    
    figure = plotly.express.bar(bar_graph_data, x = 'state', y = column_name, color = column_name)
    
    figure.show()

Converting our data from a DataFrame to a bar chart helps us better visualize the data and come to conclusions about how the data is distributed. Putting this code inside a function keeps us from having to write out the code each time we want to generate a bar graph.
